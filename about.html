GlobalHot이란?

GlobalHot은 여러 나라의 유명 커뮤니티와 뉴스 소스에서 인기글을 수집해 한 페이지에 보여주는 글로벌 트렌딩 집약 서비스입니다.

단순한 링크 목록이 아닙니다. 각 게시글을 클릭하면 미리보기 패널에서 제목·요약·썸네일을 확인한 후 원문으로 이동할 수 있습니다.

모든 데이터는 각 플랫폼의 공개 API 및 RSS를 통해 실시간으로 수집됩니다. 개인정보를 수집하지 않으며, 로그인 없이 누구나 무료로 이용할 수 있습니다.

트렌딩 점수 계산 방식

단순 추천수가 아닌, 시간의 흐름을 반영한 공식으로 순위를 계산합니다.

트렌딩 점수 = 추천수 + (댓글수 × 3) + 신선도 보너스
신선도 보너스: 최대 120점에서 시간당 2점씩 감소 (약 60시간 후 소멸). 이를 통해 오래된 인기글보다 지금 막 오르고 있는 글이 상위에 올라옵니다.

자주 묻는 질문

Q. 얼마나 자주 업데이트되나요?
페이지를 처음 열 때 자동으로 최신 데이터를 불러오며, 이후 5분마다 자동 갱신됩니다. 상단의 새로고침 버튼으로 즉시 갱신도 가능합니다.
Q. 콘텐츠를 직접 저장하나요?
아니요. 각 소스의 공개 API·RSS에서 실시간으로 가져오며, GlobalHot 서버에 콘텐츠를 저장하지 않습니다.
Q. 왜 일부 소스가 "오류"로 표시되나요?
외부 API 상태, 네트워크 환경, 일시적 제한 등으로 일부 소스가 실패할 수 있습니다. 다른 소스는 정상적으로 로딩됩니다. 새로고침으로 재시도할 수 있습니다.
Q. 개인정보를 수집하나요?
테마(다크/라이트) 설정만 브라우저 로컬 저장소에 저장하며, 서버로 전송되지 않습니다. 자세한 내용은 개인정보처리방침을 확인하세요.
