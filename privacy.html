개인정보처리방침

최종 업데이트: 2026년 3월 17일

GlobalHot(이하 "본 서비스")은 사용자의 개인정보를 중요하게 생각합니다. 본 방침은 본 서비스가 수집하는 정보와 그 사용 방법을 설명합니다.

1. 수집하는 정보

본 서비스는 개인 식별 정보를 수집하지 않습니다.

  • 회원가입 없음 – 이메일, 이름, 연락처를 수집하지 않습니다.
  • 로그인 없음 – 계정이나 패스워드를 저장하지 않습니다.
  • 로컬 저장소(localStorage): 다크/라이트 테마 설정만 사용자의 기기에 저장됩니다. 서버로 전송되지 않습니다.

2. 자동으로 수집되는 정보

Cloudflare Pages 인프라를 통해 다음의 기술 정보가 자동으로 수집될 수 있습니다.

  • IP 주소 (Cloudflare 서버 로그)
  • 브라우저 종류 및 운영체제 (User-Agent)
  • 방문한 페이지 및 시간

이 정보는 서비스 운영 및 보안 목적으로만 사용되며, 개인 식별에 활용되지 않습니다.

3. 제3자 서비스

본 서비스는 콘텐츠 제공을 위해 다음 외부 서비스의 공개 API를 사용합니다. 해당 서비스의 개인정보처리방침이 적용됩니다.

  • Reddit – 인기 게시글 API
  • Hacker News (YCombinator) – 기술 뉴스 API
  • Dev.to – 개발자 콘텐츠 API
  • GitHub – 트렌딩 저장소 API
  • Wikimedia Foundation – Wikipedia 트렌딩 API
  • はてなブックマーク – 일본 인기글 RSS
  • NHK – 뉴스 RSS
  • Microlink – URL 미리보기 생성 (클릭 시 해당 URL이 Microlink 서버에 전달됩니다)
  • AllOrigins – RSS 크로스오리진 프록시
  • Google Fonts – 웹폰트

4. 광고 (Google AdSense)

본 서비스는 Google AdSense를 통한 광고를 게재할 수 있습니다. Google은 사용자의 브라우저 쿠키를 사용하여 관심 기반 광고를 표시할 수 있습니다.

  • Google의 광고 쿠키 사용을 거부하려면 Google 광고 설정을 방문하세요.
  • Google 개인정보처리방침

5. 쿠키

본 서비스 자체는 쿠키를 사용하지 않습니다. 단, 광고 제공 목적으로 Google AdSense가 쿠키를 사용할 수 있습니다.

6. 데이터 보안

본 서비스는 다음의 보안 조치를 적용합니다.

  • 모든 통신 HTTPS 암호화
  • 보안 HTTP 헤더 적용 (X-Frame-Options, CSP, HSTS 등)
  • 사용자 데이터 서버 저장 없음

7. 아동 개인정보 보호

본 서비스는 만 14세 미만 아동으로부터 의도적으로 개인정보를 수집하지 않습니다. 해당 연령의 아동은 보호자의 동의 없이 서비스를 이용하지 않도록 권고합니다.

8. 광고 및 쿠키

본 서비스는 Google AdSense를 통해 광고를 게재할 수 있습니다. Google AdSense는 사용자의 관심사에 맞는 광고를 표시하기 위해 쿠키를 사용합니다.

  • Google의 광고 쿠키 사용으로 인해 사용자의 브라우저 활동 정보가 수집될 수 있습니다.
  • 광고 개인화를 원하지 않는 경우 Google 광고 설정에서 비활성화할 수 있습니다.
  • Google의 개인정보 처리 방식은 Google 개인정보처리방침을 참조하세요.

9. 개인정보처리방침 변경

본 방침은 서비스 변경에 따라 업데이트될 수 있습니다. 변경 시 본 페이지의 "최종 업데이트" 날짜가 수정됩니다.

9. 문의

개인정보 관련 문의는 About 페이지를 통해 연락해주세요.
